1.PWM이란?
-----------
>Pulse Width Modulation의 약자로 PWM제어라고 하며 펄스의 폭을 조정하여 부하에 전력의 크기를 조절하는 것이다.
내장된 타이머카운터를 이용하는 것이 보통이다.


2. Prescaler란?

----------------

>타이머에 공급하는 입력 클럭의 속도를 조절하는 분주기이다.
 
>예를 들어 시스템 클럭이 8MHz를 Prescaler가 2분주이면 타이머 클럭은 4MHz가 된다.     


3. Counter Period란?

---------------------

>어떤 기준점(한계점)을 설정하는 것을 말한다.

>Time 레지스터에서는 클럭 펄스가 실행될 때 레지스터 값이 1증가하며 레지스터의 값이 Counter Period 값과
같아지면 0으로 초기화 된다. 또한 이와같이 같이 Time 레지스터와 Counter Period의 값이 같아져 0으로 
초기화 되는 순간 새로운 PWM이 출력된다. 즉, LOW에서 HIGH로 변하는 Rising Edge 가 실행된다


4. Pulse란?

------------

> 매우 짧은 시간 동안에 흐르는 신호용의 약한 전류로서 제어 회로 신호에 사용한다.


5. PWM 주기와 주파수 구하기

----------------------------

>Freq = Clock Freq / Prescaler / Counter Period로 
즉 **주파수 = 기존 클럭 / Prescaler / Counter Period**이라는 공식을 통해 구할 수 있다.

>주기 = 1 / 주파수 (주기는 주파수값에 대하여 역수를 취한 것이다.) 

>예를 들어 기존의 클럭 주파수가 8MHz이고 Prescaler가 2분주이면 이것에 실제 사용 클럭은 4MHz가 된다.
Timer 레지스터는 하나의 클럭 펄스 당 값이 1씩 증가한다고 하므로, Counter Period 값을 4000으로 설정하여
Timer 레지스터 값이 4000이 되면 Rising Edge가 실행되므로 새로운 주파수를 위해서는 4000개의 
클럭 펄스가 필요하다는 것이다. 따라서 4MHz 클럭에서 4000개의 클럭 펄스가 있어야 하나의 주파수 사이클이
생기게 된다. 고로 주파수는 4MHz / 4000 = 1KH 가 되며 주기는 주파수 값에 대한 역수 이므로 1 / 1000초, 즉 
1ms가 된다.
